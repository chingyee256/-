# 给爸爸的生日网站 🎂

一个充满仪式感和温馨的生日网站，用互动蛋糕作为情感入口，展开对爸爸的祝福、感谢与回忆。

## ✨ 功能特点

- 🎂 **互动蛋糕**：点击按钮吹灭蜡烛，体验真实的生日仪式感
- 💛 **温馨设计**：温暖克制的视觉风格，不花哨但充满情感
- 📸 **回忆相册**：展示与爸爸的美好回忆
- 💌 **手写信件**：表达对爸爸的感谢与祝福
- 🎵 **背景音乐**：可选的温馨背景音乐

## 🎨 设计风格

- **主色调**：奶油白/米色背景，暖黄/金色蜡烛，深蓝/深棕强调
- **字体**：手写感标题（Caveat）+ 极简正文（Inter）
- **动画**：缓慢、克制，每个动画都在"陪他说一句话"

## 📁 文件结构

```
生日/
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # 交互脚本
├── music.mp3       # 背景音乐（需要自行添加）

```

## 🚀 使用方法

### 1. 添加照片

在 `index.html` 中找到回忆相册部分，替换照片占位符：

```html
<div class="memory-photo">
    <img src="你的照片路径.jpg" alt="回忆">
</div>
```

### 2. 添加背景音乐

将音乐文件命名为 `music.mp3` 并放在项目根目录下。

**推荐音乐类型**：
- 轻柔的钢琴曲
- 温馨的纯音乐
- 温暖的背景音乐

### 3. 自定义文字内容

- **祝福文字**：在 `script.js` 的 `showBlessingText()` 函数中修改
- **回忆文字**：在 `index.html` 的回忆相册部分修改
- **信件内容**：在 `index.html` 的 `#letter-section` 部分修改

### 4. 本地预览

直接在浏览器中打开 `index.html` 文件即可预览。

### 5. 部署上线

#### 方法一：GitHub Pages（推荐）

1. 在 GitHub 创建新仓库
2. 上传所有文件
3. 在仓库设置中启用 GitHub Pages
4. 选择主分支作为源
5. 访问 `https://你的用户名.github.io/仓库名/`

#### 方法二：其他静态网站托管

- Netlify
- Vercel
- 自己的服务器

## 🎯 使用建议

1. **照片准备**：建议准备 6-12 张照片，尺寸建议 1200x800px
2. **音乐选择**：选择轻柔、温馨的音乐，时长 2-5 分钟
3. **文字内容**：用真诚的话语，不需要华丽的辞藻
4. **测试体验**：在发布前多测试几次，确保动画流畅

## 💡 自定义建议

### 修改蜡烛数量

在 `index.html` 中调整每个蛋糕层的蜡烛数量：

```html
<div class="candles">
    <div class="candle" data-candle="1">
        <div class="flame"></div>
        <div class="smoke"></div>
    </div>
    <!-- 添加更多蜡烛 -->
</div>
```

### 修改颜色主题

在 `style.css` 的 `:root` 部分修改颜色变量：

```css
:root {
    --cream-white: #faf8f3;
    --warm-yellow: #f4d03f;
    /* 修改为你喜欢的颜色 */
}
```

### 调整动画速度

在 `script.js` 中修改时间间隔：
- 蜡烛熄灭间隔：`setTimeout(blowNext, 200)` 中的 `200`
- 文字显示速度：`setTimeout(type, 50)` 中的 `50`

## 📱 响应式设计

网站已适配：
- 桌面端（1920px+）
- 平板（768px - 1920px）
- 手机（480px - 768px）

## ❤️ 情感设计原则

- **宁可少，也不要快**：每个动画都缓慢、有节奏
- **宁可简单，也不要炫**：保持克制，突出情感
- **每一个动画，都是在"陪他说一句话"**：让每个细节都充满温度

## 📝 注意事项

1. 音乐文件需要用户交互后才能自动播放（浏览器限制）
2. 建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
3. 照片建议压缩后使用，以提高加载速度
